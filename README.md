# FreeCompute

**Community Powered Cloud Computing**

Launch powerful cloud desktops directly from your browser. Powered by donated computers and cloud infrastructure.

## ✨ Features

- 🌍 Browser-based desktop environment
- 💻 Community-powered infrastructure
- 💳 Credit-based resource allocation
- 🔐 Secure, isolated virtual machines
- 🎮 Gaming, development, and work ready
- 📦 Open source and extensible

## 📚 Stack

**Frontend**: Next.js 15 | React 19 | TypeScript | TailwindCSS v4 | shadcn/ui

**Backend**: Go | Node.js | PostgreSQL | Redis

**Desktop**: Custom WebOS | WebRTC streaming | QEMU/KVM

**Deployment**: Docker | Kubernetes | Cloudflare Tunnel
