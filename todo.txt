FreeCompute - Change Index
==========================
All planning docs live in: docs/

To make a change, find the relevant doc below, read it, then edit code.

RUNNING THE PROJECT
  Website + gateway:  ./start-website.sh
  VM host agent:      ./start-vms.sh  (FREECOMPUTE_RESOURCE_PCT=50 by default)
  Full stack:         ./run-backend.sh
  VM dry-run check:   cd host-agent && go run ./cmd/vm-setup --dry-run

LIVE SERVICES (dev)
  Gateway:   http://localhost:8080   (Go — all business logic + tunnels)
  Frontend:  http://localhost:3000   (Next.js 15 — console + WebOS)
  WebOS:     http://localhost:3000/webos

WHAT NEEDS REAL DEPENDENCIES
  QEMU VM launch        needs: qemu-system-x86_64 installed
  FFmpeg encoding       needs: ffmpeg installed + in PATH
  Tailscale mesh        needs: tailscale installed + logged in
  Stripe payments       needs: STRIPE_SECRET_KEY env
  LLM moderation        needs: FREECOMPUTE_MODERATION_LLM_URL env

VM RESOURCE ALLOCATION
  Default:   50% of host (auto-detected)
  Override:  FREECOMPUTE_RESOURCE_PCT=75  (in start-vms.sh)
  Manual:    FREECOMPUTE_VM_CPUCORES=6 FREECOMPUTE_VM_RAMGB=4 FREECOMPUTE_VM_STORAGEGB=344
  This machine (50%): 6 CPUs, 4GB RAM, 344GB disk

KEY ENV VARS
  FREECOMPUTE_TUNNEL_TOKEN     Gateway auth token (default: dev-token)
  FREECOMPUTE_DB_PATH          SQLite DB location (default: /tmp/freecompute.db)
  FREECOMPUTE_ADMIN_EMAIL      Admin account email (default: admin)
  FREECOMPUTE_ADMIN_PASSWORD   Admin password (auto-generated if unset)
  FREECOMPUTE_RESOURCE_PCT     VM resource allocation % (default: 50)
  FREECOMPUTE_GATEWAY_ADDR     Gateway listen address (default: :8080)
  FREECOMPUTE_CDN_HOSTNAME     Production CDN hostname
  FREECOMPUTE_EDGE_HOSTNAME    Production edge hostname
  FREECOMPUTE_MODERATION_LLM_URL  Optional LLM for AI moderation
